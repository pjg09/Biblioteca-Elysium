# Sistema de Biblioteca - Backend

Proyecto backend para gestión de biblioteca.

## Requisitos
- Java 17+
- Maven 3.6+

## Ejecutar el proyecto
```bash
mvn clean compile exec:java
```

## Estructura
```
src/main/java/com/biblioteca/
├── Main.java
├── modelos/
├── servicios/
├── repositorios/
└── interfaces/
````
